# Entwurf: Anfrage zur Distribution abgeleiteter Unique-Planerdaten

Status: `not-pursued` – nicht versendet, nicht zur Versendung vorgesehen.

Historie: 5M.2.8A bereitete diesen konservativen Klärungsweg vor. Der
Auftraggeber hat in 5M.2.8B ausdrücklich entschieden, keine externe
Einzelfreigabe anzufragen. Es liegt daher weder eine Antwort noch eine
Genehmigung vor. Der Text bleibt ausschließlich als Entscheidungsverlauf
erhalten.

Hello Grinding Gear Games,

we maintain a free, public, open-source Path of Exile 2 build-planning web app
published through GitHub Pages. We would like to ship a small normalized data
set for Unique-item planning. The input would be 20 exactly pinned static
Unique files from Path of Building 2; the output would contain only the
functional planner fields needed for Unique selection and comparison.

The proposed data set would contain item names, planner base/slot labels,
requirements, visible modifier expressions, structured roll ranges, variants
and legacy markers. It would not contain images, icons, flavour text, a copy
of the source files or the complete PoB2 database. It would not use runtime
APIs, hotlinks, scraping or circumvention of technical protections. It would
be generated at build time from a fixed source pin and delivered locally with
the GitHub Pages app.

The app is not sold, and the data itself would not be sold. We would include
the required non-affiliation statement and any attribution or copyright
notice you specify. The planner would clearly describe these records as
third-party planner data, not as official GGG Unique, Mod or Stat identities.

Would GGG give prior written approval for storing and publicly distributing
this reduced derived Unique planner data set with the described web app? If
so, please specify any required attribution, permitted fields, restrictions
or review process.

No data distribution will begin unless the required permissions are resolved.
